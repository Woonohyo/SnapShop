Sets URL of the image that should be loaded into this view. Note that calling this will immediately either set the cached image (if available) or the default image specified by VolleyImageView#setDefaultImageResId(int) on the view.

NOTE: If applicable, VolleyImageView#setDefaultImageResId(int) and VolleyImageView#setErrorImageResId(int) should be called prior to calling this function.

Parameters
urlThe URL that should be loaded into this ImageView.
imageLoaderImageLoader that will be used to make the request.
